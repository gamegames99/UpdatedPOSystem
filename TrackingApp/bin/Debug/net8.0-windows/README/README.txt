If you have a problem with the database follow these instructions :
1) Make sure you have a MySQL Server installed on your computer
2) (Optional) Install MySQL Workbench to configure your database server and connections
3) Find "Everything.vb" in the main directory of this project
4) Open "Everything.vb" into a notepad or any text editor
5) Find the text : "server=127.0.0.1;user=root;password=user;database=project"
6) Change the server, user, password and database name accordingly to match your database connection
7) Main directory > bin > debug > net8.0windows > Application.exe and click "Run as administrator"
